# Sales Dashboard - Power BI Project

## 📊 Overview
This project presents an interactive sales dashboard built using Power BI to analyze business performance and trends.

## 🛠 Tools Used
- Power BI
- Excel

## 🔍 Key Features
- Interactive slicers for filtering data
- Sales performance tracking
- Trend analysis for better decision-making

## 📈 Insights
- Identified top-performing regions
- Analyzed monthly sales trends
- Highlighted key business opportunities
